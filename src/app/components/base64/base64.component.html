Base64 Encoder & Decoder

Encode any text or binary data to Base64 and decode it back. Useful for embedding images, data URIs, or transferring binary data via text-only channels.

How to use

  1. Choose mode: Encode or Decode.
  2. Paste your text or Base64 string into the input box.
  3. Click Convert and copy the output.

Features

  • Full Unicode support via encodeURIComponent.
  • Error handling for invalid input.
  • Instant, client-side processing.
